HTML stands for HyperText and Markup Language

In general it consists of Text content, Markup, References to other documents
and Links to other pages

In order to manipulate the content we create, we use tags.
Tags are our way of telling the "stupid" computer, what we want it to do.

Some tags use attributes, like href or src to classify
the procedure behind that content more precisely.

Attributes can be used:
- To reference other documents (like with the image above),
- To Link to other Websites,
- To assign CSS code to certain content

Tags (or Elements) can be distinguished into Inline Elements & Block (or Container) Elements
This will become more important as the degree advances :)
